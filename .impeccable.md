# Design Context — HXFlow Docs

## Product
HXFlow 工程框架的内部技术参考文档，托管于本地/内网，供工程师（前端/后端/移动端）在浏览器中阅读。

## Audience
内部工程师，熟悉技术工具，重视效率和信息密度，在 macOS / Windows 桌面浏览器使用。

## Brand personality
工程感、精准、专业；克制的暗色调，金色作为主强调色，配衬线字体标题与等宽代码字体。

## Design tokens (hx-docs.css)
- `--gold` / `--gold-soft` — 主强调色
- `--blue`, `--teal`, `--violet`, `--orange`, `--red` — 语义色
- `--text`, `--text-muted`, `--text-faint` — 文本层级
- `--serif: Noto Serif SC` — 标题
- `--sans: Noto Sans SC` — 正文
- `--mono: JetBrains Mono` — 代码/标签

## Quality bar
内部工具，目标是「体验好的工程文档」，不追求视觉炫技，但细节要精准一致。

## Files
- `assets/hx-docs.css` — 共享样式
- `assets/hx-docs.js` — 主题切换 + 侧栏
- `docs/guide/*.html` — 各子文档页
